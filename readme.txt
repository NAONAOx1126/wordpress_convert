"BiND for WordPress theme converter" plug-in is template converter.
It converts the WordPress template created by "BiND for WebLiFE 6" into WordPress Theme files.

== Description ==

"BiND for WordPress theme converter" plug-in has the function which converts template files created by " BiND for WebLiFE 6" into WordPress Theme files.

"BiND for WordPress theme converter" has another function, we call it "BiND for WordPress DashBorad".
"BiND for WordPress DashBorad" is customized Dashboard which assists users to use their WordPress.

Function List
Convert BiND6 Template files into WordPress theme files
Customized dashboard

The plug-in converts only a template created by  " BiND for WebLiFE 6".
About the use of this plug-in, the " BiND for WebLiFE 6" product support is not covered.

== Installation ==

how to upload
　１）Upload the files to the /wp-content/plugins/WordPress_theme_converter directory.
　２）Activate the plugin through the 'Plugins' menu in WordPress.
