This is Theme Converter for HTML generated by BiND, Web Authoring Tool.

== Description ==

This plugin have you make WordPress Theme from HTML site data generated by BiND, Web Authoring Tool.

== Installation ==

1. Upload `wordpress_converter_plugin.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Upload BiND site data to anywhere in same server.
4. Point to site data directory in plugin convert setting menu.
5. Open plugin dashboard page to convert site data.
